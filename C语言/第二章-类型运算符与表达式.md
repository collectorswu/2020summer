# 第二章 类型、运算符与表达式
1. 数据类型
   ```
   char  字符型，占用一个字节
   int   整型
   float 单精度浮点型
   double 双精度浮点型

   限定符：
   short 通常16位
   long  通常32位
   类型限定符：
   signed
   unsigned
   ```

2. 常量
   ```
   整型常量：
   八进制形式：前带0
   如037
   十六进制形式：前带0x
   如0x1f
   无符号：后缀u
   long:后缀l

   浮点型常量：
   没有后缀的为：double
   后缀f为：float
   后缀l为：long double

   字符常量：
   一个字符常量是一个整数，字符在机器字符集中的数值就是字符常量的值
   如 'x'
   转义字符序列：
   如'\n'
   另外可以用'\ooo'表示任意的字节大小的位模式。
   其中，ooo表示1～3个八进制数字。
   也可'\xhh',其中hh是一个或多个十六进制数字。
   '\0',表示值为0的字符，也就是空字符（null）

   字符串常量 ""
   字符串内部表示使用一个空字符串'\0',作为串的结尾。
   'x'与"x"是不同的。
   
   枚举常量：
   例如：enum boolean {NO,YES};
   没有显示说明的情况下，enum类型中第一个枚举名的值为0,以此类推。
   如果指定了部分枚举名的值，那么未指定的枚举名将依着最后一个指定值向后递增。
   不同枚举中的名字必须互不相同。同一枚举中的不同名字可以有相同的值。
   又例：
   enum months {JAN = 1,FEB,MAR,APR,MAY,JUN,JUL,AUG,SEP,OCT,NOV,DEC};   

   ```

3. 声明
   ```
   1. 所有变量先声明后使用
   2. const 限定符指定变量的值不能被修改。
   3. 对数组而言，const限定符指定数组所有元素的值都不能被修改。
      int strlen(const char[]);
      表明函数不能修改数组元素的值。
   ```
